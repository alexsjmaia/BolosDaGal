Scripts do banco de dados do sistema Bolos da Gal.

Arquivos principais:
- database.sql: cria o banco e todas as tabelas principais
- criar-tabela-vendas.sql: cria apenas a tabela de vendas
- alterar-tabela-vendas-comanda.sql: ajusta uma tabela de vendas antiga

Use normalmente:
- database.sql em instalacoes novas
- alterar-tabela-vendas-comanda.sql apenas para atualizar banco antigo
